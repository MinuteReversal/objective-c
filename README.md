| 类型                  | 拼写                       |
| --------------------- | -------------------------- |
| 类名/函数/枚举/头文件 | UpperCamelCase             |
| 变量/方法             | lowerCamelCase             |
| 常量                  | SNAKE_CASE                 |
| --                    | lowercase_with_underscores |
| 文件名                | alllowercase               |
